Dans une série statistique on étudie le \defined{caractère} d'une
\defined{population} :
\begin{itemize}
    \item le caractère est constitué de plusieurs \defined{valeurs};
    \item l'\defined{effectif total} est égal au nombre total de
    valeurs;
    \item l'\defined{effectif} d'une valeur est égal au nombre de fois
    où la valeur apparaît dans la série.
\end{itemize}
